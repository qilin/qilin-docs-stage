    Publication

    Greetings! You are ready to distribute your game worldwide.

    All you need is to submit your store page and a game build for moderation.

    Store page moderation

    Make sure all mandatory steps are filled in before submitting the store page for moderation.

    The store page review will take for 2 – 3 business days. We will notify you every step of the moderation by email.

    When the store page is on moderation, you cannot change any settings of it.

    Game build moderation

    The store page review will take for 2 – 3 business days. We will notify you every step of the moderation by email.

    When the game build is on moderation, you cannot change any settings of it.

    To upload your game build:

    • Download Super!Utility Tool.
    • Connect the Utility tool to your admin’s account.
    • Upload your game build.
    • Send the request for the game build moderation.

    When the game build has status Approved you can set the release date and publish your the store page.

    Update the game build

    To update an already approved game build:

    • Connect Super!Utility Tool to your admin’s account.
    • Upload a new game build.
    • Send the request for the game build moderation.
