    Supported Languages

    In-game localization

    Specify in-game localization for Subtitles, Interface and Audio:

    • Interface — all in-game menu and interface displayed in this language.

    • Audio — each phrase translated and sounded in the specified language.

    • Subtitles — each phrase accompanied by subtitles in this language.

    The default language is English, and you can also add other languages.
